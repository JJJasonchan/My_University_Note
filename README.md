# 我的大学笔记仓库 📚


## 📖 项目简介 

欢迎来到我的大学大二课程学习笔记仓库！

本仓库目前主要收录了 **Python 程序设计** 与 **数据结构 (Data Structures)** 两门核心课程的学习资料。


## 🗂️ 笔记概览 

###  Python 程序设计 (Final Exam Review)
> 对应文件：`python 期末考试复习考点.md`

这份笔记是为 **期末突击** 准备的复习大纲，重点梳理了暨南大学 Python 的易混淆点和常考点.


###  数据结构 (Data Structures)
> 对应文件：`数据结构.md`

这份笔记涵盖了数据结构的**理论推导**与 **C 语言代码实现**，通过对比分析帮助理解底层逻辑：

* **线性表 (Linear List)**:
    * 顺序表 vs 链表 (时间复杂度对比，插入/删除操作的 C 实现)
    * **栈与队列**: 顺序栈/链栈，循环队列的“假溢出”处理与判空/判满逻辑
* **树与二叉树 (Trees)**:
    * 二叉树性质与存储 (二叉链表/三叉链表)
    * **核心算法**: 前中后序遍历 (递归实现)，哈夫曼树 (Huffman Tree) 的构建与编码算法
    * 树、森林与二叉树的相互转换
* **图论 (Graphs)**:
    * **存储结构**: 邻接矩阵 vs 邻接表 (逆邻接表)
    * **经典算法**: 最小生成树 (Prim/Kruskal)，最短路径 (Dijkstra/Floyd)，拓扑排序
* **查找与排序 (Search & Sort)**:
    * **查找**: 顺序、折半 (二分)、分块查找，以及 哈希表 (Hash Table) 与冲突解决
    * **十大排序**: 冒泡、快排、堆排序、归并排序等算法的时间复杂度
